# نظام النجوم - Star System

نظام متطور عالمي يشمل متجر تطبيقات، عملة مالية، خوارزميات، ومتصفح خاص. كل شيء مرتبط بالنجوم للوصول العالمي.

## الميزات
- متجر تطبيقات النجوم
- عملة النجم المالية
- خوارزميات النجم
- متصفح النجم

## التشغيل
```
npm install
npm start
```
الوصول: http://localhost:3000
